Privacy Policy for Text Downloader Extension

Last updated: July 20, 2025

Summary: The Text Downloader extension does not collect, store, or transmit any personal data or user information.

1. Information We Collect

The Text Downloader extension does NOT collect any personal information from users. We do not:

  • Collect or store personal data
  • Track user browsing habits
  • Send data to external servers
  • Use analytics or tracking services
  • Store cookies or local storage data

2. How the Extension Works

The Text Downloader extension operates entirely locally on your device:

  • It only accesses the current webpage when you click the extension button
  • It extracts visible text content from the current page
  • It creates a downloadable text file locally on your device
  • No data is transmitted to any external servers

3. Permissions Used

The extension uses the following permissions for its core functionality:

  • activeTab: To access the current webpage content when you click the extension
  • scripting: To inject scripts that extract text content from the current page

These permissions are used solely for the text extraction functionality and do not involve any data collection.

4. Data Storage

The extension does not store any data. All processing happens in real-time when you use the extension, and no information is retained after the text file is downloaded.

5. Third-Party Services

The Text Downloader extension does not integrate with or send data to any third-party services, analytics platforms, or external APIs.

6. Changes to This Privacy Policy

If we make any changes to this privacy policy, we will update the "Last updated" date at the top of this page. Continued use of the extension after any changes constitutes acceptance of the updated policy.

7. Contact Information

If you have any questions about this privacy policy or the Text Downloader extension, please contact us through the Chrome Web Store support page.

8. Compliance

This extension is designed to comply with:

  • Chrome Web Store Developer Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
Your Privacy Matters: We believe in transparency and user privacy. This extension is designed to work entirely on your device without collecting any personal information.
